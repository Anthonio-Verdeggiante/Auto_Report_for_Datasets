# Auto_Report_for_Datasets
В рамках работы с любыми датасетами неотъемлемой частью является предобработка данных. Это довольно рутинный процесс, идея автоматизации которого является весьма привлекательной задачей с точки зрения упрощения работы. Моя реализация задачи авто отчёта представлена в настоящем репозитории. Скрипт способен упростить работу как мне, так и другим :)
